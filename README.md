# MSc Project Analysis

Python application developed for an MSc project comparing Sim RT and Varian RGSC 
respiratory traces and image quality for 4DCT lung imaging.

VOI_Tool.py: batch HU-thresholding application for tumour insert volume measurement 
from DICOM 4DCT image sets (MIP and phase bin images). Built with Python 3, Tkinter, 
and Matplotlib.
